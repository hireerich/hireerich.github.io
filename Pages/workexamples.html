Java: Vehicles Project

In my Advanced OOP course I was tasked with creating a class heirarchy of vehicles. This would have a base class containing all similar attributes between vehicle types. Then new types of vehicles would be created with unique controls depending on their type.

This Project was built on throughout the entire course, each assignment adding a new feature to the Project. The vehicles collide with eachother, are able to be controlled, can be added and removed, are saved to a database when the program is closed and retreived when it is opened and run on a server so that multiple clients may add and remove vehicles.

C++: Floyd Warshall Project

During my data structures course in my last semester at NSCC, I was given a project to create a program that could read data from a text file containing distances between two cities and then parse it into maps and sets. These maps and sets would then be used to create tables that displayed these distances.

I was to then implement the Floyd Warshall Algorythm to link together cities that otherwise are not directly connected to one another. This essentially creates paths to each city from each city and gives a distance value for each trip. This was also to be output in a table. Finally, the program needed to accept two cities from the end user so the program could then use the data from using the Floyd Warshall Algorythm to show the end user every city they have to pass through to get to their desired destination.

Android: Quick Time Event

The base task was to use an AsyncTask to do a process in the background and display it with a progress bar.
